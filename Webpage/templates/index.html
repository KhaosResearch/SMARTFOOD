Near and remote sensing of agriculture: Sensor integration and connectivity

This Web application allows users to analyze geospatial data using their own shapefiles. It offers two algorithms: time series analysis of spectral indices in the selected area of interest using SIGPAC terrain separation; and an analysis of the data according to a classification label that must be included in the shapefile.
